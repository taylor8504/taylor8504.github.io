PHP（全称：PHP：Hypertext Preprocessor，即”PHP：超文本预处理器”）是一种通用开源脚本语言。

php 基本语法

php脚本以<?php 开始，以 ？>结束

PHP文件默认拓展名为”.php“

PHP 中两种在浏览器输出文本的基础指令：echo、print

（echo可以输出一个或多个字符串，print只能输出一个字符串返回值总为1。）

PHP中单行注释 // 注释内容 多行注释 /* 注释内容*/

php变量

变量是用于存储信息的容器

  • 以$开始，后跟变量名称。

  • 变量名必须以字母或者下划线字符开始。

  • 变量名只能包含字母数字字符以及下划线，不能包含空格。

  • 变量名区分大小写。

PHP四种不同的变量作用域：

local global static parameter

  • 全局变量和局部变量

    在函数外定义的变量为全局变量，它的作用域从定义处一直到文件结尾。

    函数内定义的为局部变量，它的作用域为函数定义范围内。

    函数之间作用域互不影响。

    想要在局部访问全局变量需要在变量前加 global 关键字。

  • Static作用域

    一般一个函数完成时，他的所有变量通常会被删除，如果希望摸个局部变量不被删除，只需 要在第一次声明变量时使用static关键字。每次调用该函数时，变量会保留着前一次被调用时的值。

  • 参数作用域

    通过调用代码将值传递给函数的局部变量

PHP变量类型：

类型 例
整型 integer 正数 0 负数
浮点型 flfloat 小数
字符串 string “ 123” ‘123’ <<<EOF 123 EOF（之间无需转译，）
资源类型 STDIN(键盘输入) STDOUT(屏幕输出) 打开的文件/网络
数组 array [1,2,3,4,”name”,1.33] 括号中可以放任何类型的符号
对象 object new person()

布尔 true false

EOF结束标识必须顶格独自占一行(即必须从行首开始，前后不能衔接任何空白和字符)

PHP常量

PHP中常量一旦赋值不会改变，尽量使用大写。

常量分为系统常量和自定义常量，使用defifine()定义常量，使用常量不用加$。

常量是全局变量。

预定义常量

FILE 查看当前文件绝对路径
DIR 查看当前文件所在文件夹
LINE 查看当前代码所在行数
PHP_OS 查看操作系统信息
PHP_VERSION 查看PHP版本信息
echo strlen(“输出引号中字符串的长度”)
$_SERVER[‘HTTP_USER_AGENT’] 查看浏览器信息

PHP 条件语句

条件语句用于根据不同条件执行不同动作

if语句

用于指定条件成立时执行代码

if…else 语句

在条件成立时执行一块代码，条件不成立时执行另一块代码

if…elseif….else 语句

在若干条件之一成立时执行一个代码块

Switch 语句

有选择地执行若干代码块之一

php 循环

在编写代码时，如果需要让相同的代码一次有一次重复运行，可以在代码中使用循环语句完成代码

while 循环

while 循环将重复执行代码块，直到指定的条件不成立

do…while 语句

do..while 语句会至少执行一次代码，然后检查条件，只要条件成立，就会重复进行循环

for 循环

for 循环用于您预先知道脚本需要运行的次数的情况

参数：

  • 初始值：主要是初始化一个变量值，用于设置一个计数器（但可以是任何在循环的开始被执行一次的代码）。
  • 条件：循环执行的限制条件。如果为 TRUE，则循环继续。如果为 FALSE，则循环结束。
  • 增量：主要用于递增计数器

foreach 循环

主要用于遍历数组

每进行一次循环，当前数组元素的值就会被赋值给 $value 变量（数组指针会逐一地移动），在进行下一次循环时，您将看到数组中的下一个值。
