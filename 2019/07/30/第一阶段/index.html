第一阶段：

linux基本命令

  • 关机 重启 退出
  • reboot init 6 shutdown -r 重启
  • shutdown -h now half init 0 shutdown poweroff 关机
  • logout 注销用户 exit 退出控制台
  • init 3 以纯文本模式启动 (无图形界面）

Ls ——list

命令格式：ls [参数] [目录名]

主要功能：列出目标目录中所有子目录和文件夹

常用参数：

-a 列出目录下的所有文件，包括以 . 开头的隐藏文件和文件夹

-l 列出文件的权限，所有者，文件大小等详细信息

-d 仅列出目录本身，而不是列出目录的文件数据

-h 将文件容量以轻易读的方式(GB，KB等)列出来

-R 连同子目录的内容一起（递归列出）

cd

命令格式：cd [参数] [目录]

主要功能：用于切换当前目录到其他目录

cd /usr/bin 跳转到 /usr/bin

cd ~ 跳到自己的home目录下

cd .. 返回上一层目录

cd / 切换到系统根目录

pwd

主要功能：查看当前工作目录的完整路径

cat

查看文件内容 -n 显示每行的行号

find

-name 查找具有指定文件名的文件

-perm 查找具有指定权限的文件

find /demo -perm 755 在demo目录下查找755权限的文件

find /etc -name “*.conf” 在etc下查找所有文件后缀为conf的文件

grep

查找所有文件中的某些字符串 可使用正则表达式

grep ‘hello’ .txt 在1.txt中查找hello

grep -i 使他不区分大小写

ls /etc | grep ‘pa’ 在etc中查找含有pa的文件

tar

命令格式：tar [参数] [文件]

常用参数：- c 建立新的压缩文件

-x 从压缩文件中提取文件

-z 支持gzip解压文件

-v 显示操作过程

-f 指定压缩文件

解压通常用 -zxvf 打包压缩通常用 -zcvf

touch

命令格式：touch [参数] [文件]
主要功能：创建文件更改文件时间(当前文件夹)
常用参数：
-t 使用指定的日期时间，而非现在的时间
命令示例：

touch 1.txt 创建1.txt文件

touch -t 1.txt

mkdir — Make Directory

命令格式：mkdir [参数] [目录]
主要功能：在指定位置创建指定的文件夹或目录

常用参数：
-p ：自动逐级建立文件夹

Mv——move

命令格式：mv [参数] [源文件或目录] [目标文件或目录]
主要功能：将文件重命名或将其移至一个新的目录中
常用参数：
-b ：若需覆盖文件，则覆盖前先行备份。
-f ：如果目标文件已经存在，不会询问而直接覆盖；

*Mv 1.txt 2.log * 将1.txt重命名为2.log

Mv 1.txt /etc 将1.tx t移动到/etc目录

more查看内容较多的文件 按q停止显示

less

命令格式：less [参数] [文件]
主要功能：按页来翻看文件的内容
常用参数：
-m 显示类似more命令的百分比
-N 显示每行的行号
-o 保存输出的内容

less用法和more类似，但可以利用上下键卷动文件

cp —Copy

将源文件复制至目标文件，或将多个源文件复制至目标目录
-R ：复制目录及目录内的所有项目

*cp -r ./ * 指定目录或文件 . 最后的点代表复制当前目录

cp demo /etc #复制demo.txt文件到/etc目录

cp -R /demo /opt #复制/demo目录下的所有项目到/opt目录下

whereis ls

查找某个命令或程序所在位置

-b 定位可执行文件

-s定位源代码文件

Which

搜索PATH变量中某个系统命令的位置

which pwd #搜索pwd命令所在路径

Ping

Linux ping命令用于检测主机

Ping -c 10 www.baidu.com ping百度10次

Ifconfig

查看配置网络设备

-up 启动指定网络设备/网卡
-down 关闭指定网络设备/网卡
-a 显示全部接口信息

ifconfig eth0 up 开启网卡eth0

Chown

改变文件的拥有者和群组

-R 处理指定目录以及其子目录下的所有文件

Chown admin：group 1.txt 将1.txt的用户拥有者改为admin，用户组为group

Chmod

-R 处理指定目录以及其子目录下的所有文件

权限范围

u ：目录或者文件的当前的用户
g ：目录或者文件的当前的群组
o ：除了目录或者文件的当前用户或群组之外的用户或者群组
a ：所有的用户及群组

Chmod a+x 1.txt 将1.txt文件所有的用户及群组增加执行权限

jobs

主要功能：查看当前正在运行的任务

jobs命令只看当前终端生效的，关闭终端后，在另一个终端jobs已经无法看到后台跑得程序了，此时利用ps（进程查看命令）

bg

将指定作业放到后台执行

bg %1 不带%时默认最后一个进程

fg

将指定作业放到前台执行

PS

显示当前进程状态

a 显示同意终端下的所有进程
u 指定用户的所有进程

ps -ef 显示所有进程

ps -aux|grep chat.js 将ps命令和grep结合

以下这条命令是检查java 进程是否存在：ps -ef |grep java

a: 显示所有程序

u: 以用户为主的格式来显示

x: 显示所有程序，不以终端机来区分

useradd

建立新用户

​ -d 指定用户登入时的启始目录
​ -g 指定用户所属群组
​ -m 自动建立用户的登入目录
​ -M 不要自动建立用户的登入目录
​ -u 指定用户ID
​ -r 建立系统账号
​ -s 指定用户登录后所使用的Shell

​ useradd -g username groupname #为添加的用户指定相应的用户组

Userdel

用于删除账号

-r 删除用户登入目录及目录中所有文件

Groupdel

​ 删除用户组

​ groupadd -g 101 group2 指定在GID101中添加用户组

​ groupmod -g 修改既有的GID -n修改既有的用户组名称

Passwd

用来更改使用者的密码

​ -d 删除密码
​ -f 强制执行
​ -l 停止账号使用 账号禁用 shadow中的！！双感叹号
​ -u 启动停止账号 解锁账号
​ -g 修改群组密码

​ -S 显示用户密码

Newgrp

​ 用于登入另一个群组

​ newgrp root 改变当前用户群组为root组

Usermod

​ 修改用户账号属性

​ -d 修改用户登录时的目录
​ -g 修改用户所属的群组

​ -G 修改用户所属的附加群组
​ -l 修改账户名称
​ -u 修改用户ID

​ -L 锁定账号

​ -U 解除密码锁定

​ usermod -G root bbb 将用户bbb加到root用户组

groupmod

​ 修改用户组名

​ -n 设置欲使用的群组名称

​ groupmod -n group regroup 重命名群组为regroup

Rm

​ 删除一个目录中的一个或多个文件或目录
​ -f 强制删除，不会出现警告消息
​ -r 递归删除，常用于目录删除
​ -v 详细显示进行的步骤

​ rm 删除文件

​ rmdir 删除文件夹 -p 为逐级删除

其他

head 显示文件的前几行

tail 显示文件的后几行 动态显示

ctrl +z 将正在执行的命令放到后台，并暂停

ctrl +x 后台运行

ctrl +d 停止
