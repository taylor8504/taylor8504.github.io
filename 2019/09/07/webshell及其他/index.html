什么是shell

shell是系统的用户界面，提供了用户与内核进行交互操作的一种接口。接收用户输入的命令并把它送入内核去执行。

什么是webshell

黑客在入侵了一个网站后，通常会留下一个asp、aspx、jsp、php等脚本后门文件与网站web服务器进行交互，得到一个命令执行环境，以达到控制网站服务器的目的。这个后门文件就是所谓的webshell。

一句话木马介绍

一句话木马是webshell的一种，由于这类webshell代码比较少，往往只有一行代码，所以就被称为一句话木马。

一句话木马虽然代码简单，但是结合中国菜刀等wenshell管理工具，他们的功能却是非常强大。

一句话木马举例  REQUEST既能接收POST和GET数据，日志中不记录POST提交数据 只记录GET数据

Mysql 日志写webshell

Mysql 每执行一个sql语句，就会把sql语句写入到文件日志中，但是默认是关闭的，这个选项是关闭的。因此，需要去开启这个选项，在设置日志路径，日志路径设置为网站目录，日志文件名字为.php结尾的文件。

Set global general_log = ‘no’

Set global general_log_file = ‘c:phpstudy/phptutrial/www/500.php’
